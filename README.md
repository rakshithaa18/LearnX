LearnX – AI Powered Learning Platform

LearnX is an AI-powered educational platform that helps users learn concepts step-by-step through explanations, code examples, and interactive quizzes with multilingual and voice support.

##  Problem Statement
Many learners struggle with:
* Unstructured explanations
* Lack of practical examples
* Limited practice and feedback
* Language barriers

##  Solution
LearnX provides a structured learning flow:
Understand → Code → Practice
* AI-generated explanations
* Code examples
* Interactive quizzes
* Voice-based learning
* Multi-language support


## Features
*  Step-by-step concept explanation
*  Code implementation
*  Quiz with score evaluation
*  Multi-language support
*  Read Aloud (Text-to-Speech)
*  Voice Input (Speech-to-Text)
*  Download content as PDF
*  YouTube learning resources
*  Learning history tracking
